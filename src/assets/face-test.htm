Alternative site with Avatar generator



Face.co lets to create custom avatars. The Avatars Generator is based on SVG (Scalable Vector Graphic), which is supported by all modern browsers and does not depend on screen resolutions. As a result your visitors can download their avatars as SVG file or as PNG one (2 size options) which is converted from vector graphic parts. The integration of the SVG Avatars creator is quite simple too. This site has responsive interface and doesn’t use any usual images, like JPGs or PNGs (except for downloading), only vectors, which give us clean and perfect look on any devices including Retina.
